# 技术栈选型设计

## 1 后端技术栈

| 技术 | 选型 | 版本 | 选择原因 |
|:---|:---|:---|:---|
| 编程语言 | Java | JDK 11 | LTS版本，Spring Boot 2.7官方支持，语法够用 |
| 开发框架 | Spring Boot | 2.7.18 | 团队熟悉，生态成熟，文档丰富，2.x稳定 |
| ORM框架 | MyBatis-Plus | 3.5.5 | 在MyBatis基础上封装CRUD，减少手写SQL，自带分页 |
| 数据库连接池 | HikariCP | Spring Boot内置 | 性能最优，无需额外配置 |
| 缓存 | Redis | 6.2+ | 文档已确定，StringRedisTemplate操作简单 |
| 安全框架 | Spring Security + JWT | Spring Boot内置 | 集成JWT无需额外引入复杂框架 |
| 消息队列 | 本地阻塞队列 + 线程池 | JDK内置 | 视频转码异步处理，5人团队无需引入MQ中间件 |
| 视频处理 | FFmpeg | 5.1+ | 文档已确定，命令行调用最简单 |
| 直播服务器 | SRS | 4.0+ | 文档已确定，开箱即用 |
| 构建工具 | Maven | 3.8+ | 团队熟悉，Spring Boot默认 |
| JSON处理 | Jackson | Spring Boot内置 | 无需额外引入 |
| 日志框架 | SLF4J + Logback | Spring Boot内置 | 开箱即用 |
| API文档 | Swagger (SpringDoc OpenAPI) | 1.7+ | 注解方式自动生成，比手写文档快 |
| 工具库 | Hutool | 5.8+ | 国产工具库，覆盖常用工具方法，减少重复代码 |

### 1.1 关键依赖清单 (pom.xml)

```xml
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>2.7.18</version>
</parent>

<dependencies>
    <!-- Web -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
    <!-- WebSocket (弹幕/直播信令) -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-websocket</artifactId>
    </dependency>
    <!-- Security + JWT -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-security</artifactId>
    </dependency>
    <dependency>
        <groupId>io.jsonwebtoken</groupId>
        <artifactId>jjwt</artifactId>
        <version>0.9.1</version>
    </dependency>
    <!-- MySQL -->
    <dependency>
        <groupId>mysql</groupId>
        <artifactId>mysql-connector-java</artifactId>
        <version>8.0.33</version>
    </dependency>
    <!-- MyBatis-Plus -->
    <dependency>
        <groupId>com.baomidou</groupId>
        <artifactId>mybatis-plus-boot-starter</artifactId>
        <version>3.5.5</version>
    </dependency>
    <!-- Redis -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-data-redis</artifactId>
    </dependency>
    <!-- 连接池 (Lettuce是Spring Boot默认) -->
    <!-- 文件上传 -->
    <dependency>
        <groupId>commons-fileupload</groupId>
        <artifactId>commons-fileupload</artifactId>
        <version>1.5</version>
    </dependency>
    <!-- Swagger -->
    <dependency>
        <groupId>org.springdoc</groupId>
        <artifactId>springdoc-openapi-ui</artifactId>
        <version>1.7.0</version>
    </dependency>
    <!-- Hutool -->
    <dependency>
        <groupId>cn.hutool</groupId>
        <artifactId>hutool-all</artifactId>
        <version>5.8.23</version>
    </dependency>
    <!-- 测试 -->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>
```

## 2 前端技术栈

| 技术 | 选型 | 版本 | 选择原因 |
|:---|:---|:---|:---|
| 框架 | Vue.js | 3.3+ | 文档已确定，Composition API代码更简洁 |
| 构建工具 | Vite | 4.4+ | 启动快，热更新快，比Vue CLI配置少 |
| UI组件库 | Element Plus | 2.3+ | Vue 3官方推荐，组件全，文档中文 |
| 状态管理 | Pinia | 2.1+ | Vue 3官方推荐，比Vuex API更简单 |
| 路由 | Vue Router | 4.2+ | Vue 3官方配套 |
| HTTP客户端 | Axios | 1.5+ | 团队熟悉，拦截器配置方便 |
| 视频播放器 | DPlayer | 1.27+ | 开源，支持弹幕层叠加，API简单 |
| 直播播放器 | flv.js | 1.6+ | 纯JS解析FLV，配合SRS的HTTP-FLV输出 |
| 弹幕引擎 | 自定义Canvas层 | - | DPlayer自带弹幕功能，无需额外引入 |
| 图标 | Element Plus自带图标 | - | 无需额外引入Icon库 |
| CSS预处理 | 原生CSS + CSS变量 | - | 5人团队无需引入Less/Sass，降低复杂度 |
